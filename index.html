Web Browser API's

Canvas API

El API de Canvas te permite dibujar elementos 2d en el navegador, asi como animarlos. Se usa en graficos para juegos en la web, visualizacion de datos, manipulacion de fotografias, etc...

Audio API

La API de Audio Web provee un sistema poderoso y versatil para controlar audio en la Web

Web Animation API

La API de Animacion epresenta un reproductor de animación y proporciona controles de reproducción y una línea de tiempo para un nodo o una fuente de animación.

Fetch API

La API Fetch proporciona una interfaz para recuperar recursos (incluso a través de la red).

Notification API

La API Push otorga a las aplicaciones web la habilidad de recibir mensajes enviados a ellas desde un servidor, sin importar si la aplicación está o no en primer plano o cargada
